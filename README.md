# dotfiles

My personal configuration files. Managed with GNU Stow.

## Setup

git clone https://github.com/s1gint-z3r0/dotfiles ~/.dotfiles
cd ~/.dotfiles
stow zsh vim git

## Contents

- `.zshrc` , Zsh config
- `.vimrc` , Vim config
- `.config/git/config` , Global git settings
- `scripts/setup.sh` , Bootstrap
